data-role page 设置你的页面为mobile的样式就靠它了，放在body的下面是mobile的容器<div data-role="page"></div>
header 用来设置你的页面的头部使用方法和page类似，<div data-role="header"></div>
footer 用来设置你的页面的底部使用方法和header类似，<div data-role="footer"></div>
content 用于设置页面的内容部分<div data-role="content"></div>
controlgroup 把几个标签设置成一个组，这个一般会配合data-type="horizontal"一起使用
fieldcontain 使用它会给这个标签定义一下下划线和一个下边距
navbar 从字面的意思你就可以想到它的用法，对就是用来设置你的导航菜单的
listview 以列表的样式显示（这个样式是预先定好的）这个一般会配合data-inset="true"使用。例如<ul data-role="listview" data-inset="true"></ul>
list-divider 这个是用在li上面的用于设置成众多li的标题，并且这个li中不能存在a标签
button 这个属性可以使你的标签样式呈现出和按钮相同的外观
